# Screenshots

Referenced from the project [README.md](../README.md). **Excluded from the WordPress.org release ZIP** by `.github/workflows/release.yml`, so adding heavy images here does not bloat the plugin distribution.

Naming convention so the README can pick them up:

| File | What it shows |
|---|---|
| `01-records-list.png` | Records tab with live filters, integrity column and CSV export. |
| `02-settings.png` | Settings tab with editable texts, version control, retention, email options and uninstall control. |
| `03-order-metabox.png` | Order edit screen with the consent metabox (summary, integrity badge, certificate button). |
| `04-orders-column.png` | Consent column on the orders list. |
| `05-certificate.png` | Printable A4 certificate (the browser saves it as PDF). |

Recommended size: 1600×1000 (or any 16:10 ratio) PNG, < 300 KB each.
